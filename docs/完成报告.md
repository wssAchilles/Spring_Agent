## 三、兼容性问题解决方案

### 3.1 问题描述

| 组件 | 原版本 | 问题 |
|------|--------|------|
| spring-ai | 1.1.0 | 2.0.0 需 Spring Boot 4.0（Spring Framework 7.0） |
| spring-ai-alibaba-dashscope | 1.1.2.2 | 2.0.0-M1.1 需 Spring Framework 7.0 |
| Spring Boot | 3.5.8 | 不兼容 Spring Framework 7.0 |

### 3.2 解决方案

**核心策略**：将 DashScope SDK 替换为 OpenAI 兼容模式，消除对 `spring-ai-alibaba-dashscope` 的直接依赖。

| 替换项 | 原实现 | 新实现 |
|--------|--------|--------|
| Chat 模型 | `DashScopeChatModel` | `OpenAiChatModel` + DashScope 兼容端点 |
| Embedding 模型 | `DashScopeEmbeddingModel` | `OpenAiEmbeddingModel` + DashScope 兼容端点 |
| Rerank 模型 | `DashScopeRerankModel` SDK | 自定义 `DashScopeRerankClient` HTTP 客户端 |
| Baidu 搜索 | `spring-ai-alibaba-starter-tool-calling-baidusearch` | 已移除（未使用） |
| Agent 框架 | `agent-framework` + `graph-core` | **保留**（ReactAgent 核心编排） |

**兼容端点**：`https://dashscope.aliyuncs.com/compatible-mode/v1`

### 3.3 验证结果
