### H1. TokenUsage.add() 非线程安全

**文件**：`TokenUsage.java:19-22`

**当前代码**：
```java
public void add(long promptTokens, long completionTokens) {
    this.promptTokens += promptTokens;       // 非原子
    this.completionTokens += completionTokens;
}
```

**修复方案**：字段改为 `AtomicLong`，add() 使用 `addAndGet()`。

**验收标准**：多线程 1000 次 add() 后值准确
